# 画像フォルダ（テーマ別）

投稿の `theme` に対応するサブフォルダに、デュクラス大阪の実写真を入れてください。
スクリプトは投稿テーマに合うフォルダから1枚を自動で選びます（直前と同じ画像は避けます）。

対応テーマ（フォルダ名）：

- `dress/` … ドレス・試着・衣装
- `food/` … 料理・試食・コース
- `chapel/` … チャペル・挙式・誓い
- `venue/` … 会場・披露宴・内観・装飾
- `photo/` … フォトスポット・前撮り・映え
- `fair/` … ブライダルフェア・相談会・見学の様子
- `hospitality/` … おもてなし・スタッフ・サービス
- `access/` … 外観・アクセス・立地

推奨：横長（4:3 / 16:9）・1枚あたり 10MB 未満・jpg/png/webp。
ファイル名は自由（例：`dress-01.jpg`）。

※ 画像付き投稿を有効にするには、別途リポジトリ設定が必要です（公開URLでの配信設定）。
　 まずはテキスト＋ボタンで運用を開始し、準備でき次第ここに画像を追加します。
